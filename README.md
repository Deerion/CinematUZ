## 🚀 Kluczowe funkcjonalności
* 🔐 **Autoryzacja:** Rejestracja i logowanie (Firebase Authentication).
* 🗺️ **Mapa Kin:** Integracja z Google Maps API – lokalizacja najbliższych kin.
* 🌓 **Personalizacja:** Obsługa trybu jasnego i ciemnego.
* 📶 **Grupy lokalne:** Tworzenie grup znajomych w pobliżu przez Bluetooth.
* 🔍 **Wyszukiwarka:** Integracja z zewnętrznym API (TMDB).
* 📚 **Biblioteka:** Lokalna baza danych Room (Obejrzane / Do obejrzenia).
* 🔔 **Powiadomienia:** Push notifications (Firebase Cloud Messaging).
* 🎲 **Maszyna losująca:** Wybór filmu poprzez potrząśnięcie telefonem (Akcelerometr).
* 📲 **Udostępnianie:** Przesyłanie propozycji przez Bluetooth.
* 📄 **Raporty:** Eksport rankingu filmów do pliku PDF.

## 🛠️ Technologie i narzędzia
* **IDE:** Android Studio
* **Język:** Java
* **Baza danych:** Room, Firebase
* **VCS:** GIT
* **Inne:** Google Maps SDK, TMDB API, Bluetooth Adapter.
